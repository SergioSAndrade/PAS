Projeto desenvolvido por Sérgio Andrade e Uóston Santos para a disciplina de Projetos Avançados de Sistemas, do curso de Sistemas de Informação, na Universidade do Estado da Bahia(UNEB), ministrada pelo professor Eduardo Jorge.

Esse projeto tem o objetivo de mostrar o funcionamento do padrão de projeto 'Cache Management'. O sistema simula uma máquina de ponto, onde colaboradores de uma empresa digitam seu 'ID' ao chegarem para trabalhar.

A pasta 'CacheManagementEx' possui um exercício para deixar claro o funcionamento e a aplicabilidade do padrão. Nesse sistema, cada vez que um colaborador digita seu 'ID', o sistema busca no servidor as informações referentes a ele. O que tem que se fazer é aplicar o padrão para que exista uma cache local onde serão armazenados as informações dos colaboradores que utilizaram a máquina de ponto mais recentemente, e com isso não seja necessário ir buscar as informações no servidor todas as vezes que for digitado um 'ID'. Mais informações e instruções do exercício estão no arquivo Atividade.pdf.

A pasta 'CacheManagement' possui o sistema completo, isto é, já com o padrão de projeto aplicado.

O arquivo DiagramaCMEx.png é o diagrama de classes do sistema sem o padrão aplicado, e o arquivo DiagramaCMEx.png é o diagrama de classes do sistema com o padrão.
